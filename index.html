What is CSS?

  1. CSS stands for Cascading Style Sheet

  2. CSS describes how HTML elements are to be displayed on screen,paper ,or in other media

  3. CSS is not necessary.CSS saves a lot of work It can control the layout of multiple web pages all at once

  4. External stylesheet are stored in CSS files

CSS Can be added to HTML document in 3 ways


  • Inline- by using the style inside HTML element1
  • Internal- by using element in the head section
  • The most common way add CSS .is to keep the styles in external CSS files.however.in this tutorials we will use inline and internal styles. because this is easier to demonstrate. and easier for you to try it yourself

When comparing CSS class vs ID,the difference is that CSS class applies a style to multiple elements.ID,on the other hand,applies a style to one unique element.ID is also special in that you can use a special url .to link directly to an element and it is used by javascript.

In CSS selectors are used to target a specific elements or range of element on a webpage Once an element has targeted,a style or set of style can applied to the elements

There is a wide range of selectors available .Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied
